# Pode (Mac) — podcasts, transcribed

A native macOS SwiftUI port of the `pode` design — liquid glass on warm paper, with real RSS subscription, AVFoundation playback, OpenAI Whisper transcription, and Claude AI analysis. SwiftData for persistence. macOS 14+.

## Open and run

The Xcode `.xcodeproj` is generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `project.yml`.

```bash
cd /Users/steve/Dev/PodeMac
open Pode.xcodeproj
# Select "My Mac" as run destination, ⌘R
```

If Xcode complains about a missing CoreSimulator framework on first launch:
```bash
sudo xcodebuild -runFirstLaunch
```

To regenerate the project after editing `project.yml`:
```bash
xcodegen
```

## Setup

1. Run the app
2. Click the gear in the sidebar footer → **Settings**
3. Paste your **OpenAI API key** (Whisper) and **Anthropic API key** (Claude)
4. Choose Whisper model (`whisper-1` or `gpt-4o-transcribe`/`gpt-4o-mini-transcribe`)
5. Choose Claude model (Haiku 4.5 / Sonnet 4.6 / Opus 4.7)

## What's real

| Capability | Implementation |
|---|---|
| Browse / Search | iTunes Search API (`itunes.apple.com/search`) and top-podcasts feed |
| Subscribe | RSS feed parsed natively with `XMLParser` — no CORS proxy needed |
| Audio playback | `AVPlayer` w/ periodic time observer; position persists every tick to SwiftData |
| Download | `URLSession.download` saves audio to Application Support; played from local file |
| Transcribe | OpenAI `/v1/audio/transcriptions` with `verbose_json` for segment timestamps |
| AI summary / takeaways / concepts | Anthropic `/v1/messages`; structured JSON output |
| Ask | Anthropic, citing line indices and timestamps you can click to seek |
| Knowledge galaxy / timeline | Real concepts from real AI runs, clustered Editorial / Mind / Body / Craft |
| Highlights | Bookmark current transcript line during playback |
| Persistence | SwiftData (`Show`, `Episode`, `TranscriptLineModel`, `Highlight`, `Concept`, `AppSettingsRecord`) |

## Design fidelity

- **Liquid glass**: `.ultraThinMaterial` for sidebar/panels, `.ultraThickMaterial` for the player dock; warm-paper canvas with two terracotta blooms layered behind
- **Type**: SwiftUI's `.serif` design (New York) for editorial moments, default sans for body, `.monospaced` for meta — close cousins of Fraunces / Inter Tight / JetBrains Mono without the bundling tax
- **Brand**: terracotta `#d06a3a` accent (configurable in Settings); the italic terracotta moment shows up in the wordmark, the greeting, and "Your *canon*."
- **Sidebar**: 244 px, native sidebar material, custom item rows with the brand pip on Knowledge
- **Player dock**: bottom-anchored glass dock with cover, transport, scrubber, live transcript ticker; expanding transcript drawer

## Notes

- Anthropic + OpenAI keys are stored in SwiftData on disk locally. The app sandbox has `network.client` so API calls work; the keys never leave your machine except to those two endpoints.
- Audio downloads land in `~/Library/Containers/studio.steve.pode/Data/Library/Application Support/Pode/Audio/`.
- The same warm-paper / terracotta design system as the web prototype, but everything that was a CORS-proxied web call is now a direct native `URLSession` call.
