# pode website

Source for [mhplala.github.io/pode-mac](https://mhplala.github.io/pode-mac/).

Static site — single `index.html` with bilingual (EN / 中) string swap via JS.
No build step, no framework. Edit + commit + push.

To preview locally:
```
cd docs && python3 -m http.server 8000
# → http://localhost:8000
```
